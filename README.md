# Awesome Service Boundaries

## Introduction

A curated list of resources on the topic of "Service Boundaries", how to design them, what are the
common anti-patterns and misconceptions.

"Service Boundaries" refers to the responsibility being given to a service and how to enforce,
document and evolve them.
